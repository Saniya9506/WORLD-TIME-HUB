# 🌍 World Time Hub

A modern and interactive **World Time Hub** built using **HTML, CSS, and JavaScript** that allows users to compare real-time clocks across different countries.

## ✨ Features

- 🕒 Live Local Clock
- 📅 Current Date & Day
- 🌍 Compare Time Between Any Two Countries
- 🌐 400+ Worldwide Time Zones
- 🔍 Searchable Country Dropdown
- ⚡ Live Time Updates Every Second
- ⏳ Accurate Time Difference Calculation
- 🎨 Modern Glassmorphism UI
- 📱 Responsive Design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Intl.DateTimeFormat API

---

## 🚀 How It Works

This project uses the **Intl.DateTimeFormat API** to fetch the current time for different time zones worldwide. Users can select any two countries and instantly view:

- Current local time
- Live clock updates
- Time difference
- Which country is ahead or behind
